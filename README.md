# NH Capital Monorepo

Private monorepo for NH Capital investment infrastructure.

## Structure

```
nhc-capital/
├── models/          # Trading models (NHL, Polymarket, etc.)
├── pipelines/       # Data ingestion & processing pipelines
├── analysis/        # Ad-hoc analysis & research
├── lib/             # Shared utilities & database helpers
├── sql/             # Database migrations & schemas
└── config/          # Configuration files
```

## Stack

- **Python** — primary language
- **PostgreSQL 17** — data storage
- **GitHub Actions** — CI/CD (TBD)

## Team

NH Capital — 5-person investment team + NHC (AI operator)
